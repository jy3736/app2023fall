Final Project - IoT-Enabled Traffic Light System Emulator and Remote Controller

App Development Practice Final Project

You will be tasked with developing both the IoT-Enabled Traffic Light System emulator and controller. Both will be implemented using App Inventor 2. The emulator is designed to replicate a real traffic light system that can be controlled remotely. It consists of only two sets of traffic lights (Red/Yellow/Green) and a text field for inputting an ID. The remote controller includes all the components of the emulator and features eight buttons to transmit specific commands. For further details, please consult the attached animation figures.

Remote Controller for IoT-Enabled Traffic Light System

You will be asked to develop a remote controller using either App Inventor 2 or Node-RED for an IoT-enabled Raspberry Pi Pico W traffic light system.

Your UI should incorporate the following components:

  • MQTT Magic Prefix Text Field: Enter the magic prefix to identify your device for remote command sending and receiving.
  • Animated Traffic Lights: These lights synchronize with the actual traffic lights controlled by the Raspberry Pi Pico W, providing real-time feedback.
  • Remote Control Buttons: Eight buttons configure the traffic light system for various operations.

Remote Control Button Commands:

  1. Straight Traffic: Prioritize straight traffic flow. 
  1. Horizontal Traffic: Prioritize horizontal traffic flow. 
  1. Flash Red Light: Flash the red light for a specific duration and interval.
  1. Flash Yellow Light: Flash the yellow light for a specific duration and interval.
  1. Traffic Light Mode 1: Activate pre-defined traffic light sequence 1.
    • Continuously repeat:
      • Red for 10 seconds (R, 10s)
      • Red flashing yellow for 0.5 seconds (R/Y, 0.5s)
      • Green for 3 seconds (G, 3s)
      • Yellow for 0.5 seconds (Y, 0.5s)
  1. Traffic Light Mode 2: Activate pre-defined traffic light sequence 2.
    • Continuously repeat:
      • Red for 5 seconds (R, 5s)
      • Red flashing yellow for 0.5 seconds (R/Y, 0.5s)
      • Green for 4 seconds (G, 4s)
      • Yellow for 0.5 seconds (Y, 0.5s)
  1. Change Light Direction: Switch between north-south and east-west traffic light control.
  1. Christmas Mode: Activate a festive light pattern on the traffic lights.
